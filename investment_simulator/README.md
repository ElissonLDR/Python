# 💰 Simulador de Investimentos

Aplicação web desenvolvida em Python (Flask) para simulação de investimentos com juros compostos e aporte mensal.

O sistema permite calcular a evolução de um investimento ao longo do tempo e visualizar o crescimento através de gráfico dinâmico.

---

## 🚀 Tecnologias Utilizadas

- Python 3
- Flask
- HTML5
- CSS3
- JavaScript (Fetch API)
- Chart.js

---

## 📌 Funcionalidades

- Cálculo de juros compostos com aporte mensal
- Simulação mês a mês
- Retorno em JSON
- Atualização dinâmica sem recarregamento de página
- Gráfico de evolução do investimento
- Interface responsiva simples

---

## 🧠 Lógica Aplicada

O cálculo é feito mês a mês:

- Aplicação de juros sobre o saldo atual
- Soma do aporte mensal
- Armazenamento do histórico em lista de dicionários
- Retorno estruturado em JSON
