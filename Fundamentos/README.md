# 📚 Fundamentos de Python

Esta pasta contém exercícios e exemplos práticos para aprendizado progressivo da linguagem Python.

Cada arquivo aborda um nível diferente de conhecimento, começando do básico até conceitos intermediários e aplicação prática.

---

## 📂 Estrutura dos Arquivos

### 01_basico_python.py

Conceitos iniciais da linguagem:

- Variáveis e tipos de dados
- Operadores matemáticos
- Condicionais (if / elif / else)
- Listas
- Loops (for / while)
- Funções
- Dicionários
- Tratamento de erros
- Estrutura principal (`if __name__ == "__main__"`)

Objetivo: Entender a base da linguagem.

---

### 02_funcoes_mais_usadas.py

Funções e métodos nativos mais utilizados no dia a dia:

- len()
- type()
- int(), float(), str()
- max(), min(), sum()
- sorted()
- range()
- enumerate()
- zip()
- any() e all()
- map() e filter()
- Métodos de string
- Métodos de lista

Objetivo: Conhecer ferramentas essenciais da linguagem.

---

### 03_funcoes_avancadas.py

Conceitos intermediários:

- Funções como objetos
- Funções dentro de funções
- Higher-order functions
- Decorators
- Generators (yield)
- List comprehension avançada
- Dictionary comprehension
- Try / Except / Else / Finally

Objetivo: Evoluir o pensamento lógico e funcional.

---

### 04_sistema_estoque_simples.py

Sistema prático completo em Python puro:

- Menu interativo
- Cadastro de produtos
- Listagem
- Atualização
- Remoção
- Uso de lista de dicionários
- Estrutura organizada em funções

Objetivo: Aplicar lógica de programação em um mini sistema real.

---

## ▶ Como Executar os Arquivos

1. Abra o terminal na pasta `Fundamentos`.

Exemplo:

2. Execute o arquivo desejado:
python 01_basico_python.py

Ou qualquer outro:
python 04_sistema_estoque_simples.py


---

## 🎯 Objetivo Geral

Esta pasta funciona como laboratório de prática para:

- Consolidar fundamentos
- Evoluir para nível intermediário
- Criar disciplina de organização de código
- Construir base sólida antes de avançar para projetos maiores
